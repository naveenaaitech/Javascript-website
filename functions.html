JavaScript Functions

1. Function Declaration

  • Declared using the function keyword.
  • Can be called anywhere in the code (hoisted).
Example:

function greet() {
  console.log("Hello, World!");
}

greet();
Output Preview:
Hello, World!

2. Function Expression

  • Stored inside a variable.
  • Not hoisted (must be declared before use).
Example:

const add = function(a, b) {
  return a + b;
};

console.log(add(5, 3));
Output Preview:
8

3. Return Statement

  • return sends a value back from a function.
  • Without return, the function returns undefined.
Example:

function square(num) {
  return num * num;
}

console.log(square(4));
Output Preview:
16

4. Parameters & Default Parameters

  • Functions can take parameters (inputs).
  • Default values can be provided if no argument is passed.
Example:

function welcome(name = "Guest") {
  console.log("Welcome, " + name);
}

welcome("Alice");
welcome();
Output Preview:
Welcome, Alice
Welcome, Guest

5. Arrow Functions

  • Introduced in ES6.
  • Shorter syntax for writing functions.
  • Does not have its own this binding.
Example:

const multiply = (a, b) => a * b;

console.log(multiply(4, 5));
Output Preview:
20

6. Callback Functions

  • A callback is a function passed as an argument to another function.
  • Used when we want a function to run after another function finishes.
Example:

function displayResult(result) {
  console.log("Result: " + result);
}

function calculate(a, b, callback) {
  let sum = a + b;
  callback(sum);
}

calculate(5, 10, displayResult);
Output Preview:
Result: 15

7. Anonymous Functions

  • An anonymous function is a function without a name.
  • Often used as arguments in other functions (like callbacks or event handlers).
  • Cannot be called directly since it doesn’t have a name.
Example:

// Using an anonymous function as a callback
setTimeout(function() {
  console.log("This message appears after 2 seconds");
}, 2000);
Output Preview:
This message appears after 2 seconds
