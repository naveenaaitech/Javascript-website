JavaScript Strings

1. What is a String?

  • A string is a sequence of characters enclosed in quotes.
  • Can use single, double, or backticks (`) for template literals.
Example:

const greeting = "Hello, World!";
console.log(greeting);
Output Preview:
Hello, World!

2. Accessing Characters

  • Use index numbers starting from 0.
  • Use charAt(index) method as well.
Example:

console.log(greeting[0]);        // H
console.log(greeting.charAt(7)); // W
Output Preview:
H
W

3. String Properties

  • length returns the number of characters.
Example:

console.log(greeting.length);
Output Preview:
13

4. Common String Methods

  • toUpperCase() / toLowerCase()
  • trim(): removes whitespace
  • slice(start, end) / substring(start, end)
  • replace(old, new)
  • includes(substring)
  • split(separator)
  • concat(str1, str2)
Example:

console.log(greeting.toUpperCase());
console.log(greeting.toLowerCase());
console.log("   Hello   ".trim());
console.log(greeting.slice(0,5));
console.log(greeting.substring(7,12));
console.log(greeting.replace("World","Alice"));
console.log(greeting.includes("Hello"));
console.log(greeting.split(","));
console.log("Hello".concat(" ","World"));
Output Preview:
HELLO, WORLD!
hello, world!
Hello
Hello
World
Hello, Alice!
true
["Hello"," World!"]
Hello World
