JavaScript Variables

What are Variables?

  • Variables are containers for storing data values.
  • They allow us to reuse and manipulate data in programs.
Example:

let name = "John";
let age = 25;
console.log(name, age);
Output Preview:
John 25

var, let and const

  • var → old way, function-scoped.
  • let → block-scoped, modern and flexible.
  • const → block-scoped, but cannot be reassigned.
Code:

var city = "Delhi";
let country = "India";
const pi = 3.14;

console.log(city);
console.log(country);
console.log(pi);
Output Preview:
Delhi
India
3.14

Re-declaration & Re-assignment

  • var can be re-declared and re-assigned.
  • let cannot be re-declared in same scope, but can be reassigned.
  • const cannot be re-declared or reassigned.
Code:

var x = 10;
var x = 20;  // valid

let y = 15;
y = 30;        // valid

const z = 50;
z = 100;      // error
Output Preview:
20
30
Error: Assignment to constant variable

Variables Without Keyword

  • You can declare a variable without var, let, or const.
  • It becomes a global variable automatically.
  • This is not recommended because it can overwrite global variables and cause bugs.
  • In 'use strict' mode, it will throw an error.
Code:

name = "Alice";  // no keyword, becomes global
console.log(name); // Alice

'use strict';
age = 25;        // ReferenceError in strict mode
console.log(age);
Output Preview:
Alice
ReferenceError: age is not defined

Hoisting

  • var declarations are hoisted to the top and initialized with undefined.
  • let and const are hoisted but remain in a temporal dead zone (TDZ) until initialization.
  • Variables declared without keyword are treated as global and hoisted like var (if not in strict mode).
Code:

// var hoisting
console.log(a);  // undefined
var a = 10;

// let & const hoisting
// console.log(b); // ReferenceError: Cannot access 'b' before initialization
let b = 20;

// const c; // ❌ Error: Missing initializer
const c = 30;

// Variable without keyword
console.log(d); // undefined (hoisted as global)
d = 40;
console.log(d); // 40
Output Preview:
undefined
40
