JavaScript DOM

1. What is DOM?

  • DOM (Document Object Model) represents HTML elements as objects.
  • Allows JavaScript to interact with web page elements.
Example Preview:
DOM allows JavaScript to read and modify content and styles.

2. Accessing Elements

  • getElementById
  • getElementsByClassName
  • querySelector / querySelectorAll
HTML Example:

<h2 id="title">Sample Title</h2>
<ul>
  <li class="item">Item 1</li>
  <li class="item">Item 2</li>
  <li class="item">Item 3</li>
</ul>
JavaScript Example:

const title = document.getElementById("title");
const items = document.getElementsByClassName("item");
const firstItem = document.querySelector(".item");
console.log(title, items, firstItem);
Output Preview:
<h2 id="title">Sample Title</h2>
HTMLCollection of items
First element with class "item"

3. Changing Content

  • Use innerHTML or textContent.
HTML Example:

<h2 id="title">Old Title</h2>
JavaScript Example:

title.innerHTML = "New Title!";
console.log(title.textContent);
Output Preview:
New Title!

4. Changing Styles

  • Modify CSS properties directly using style.
HTML Example:

<h2 id="title">Styled Title</h2>
JavaScript Example:

title.style.color = "red";
title.style.backgroundColor = "yellow";
Output Preview:
Sample Title with updated styles

5. Event Listeners

  • Use onclick or addEventListener for interactions.
HTML Example:

<button id="btn">Click Me</button>
JavaScript Example:

const btn = document.getElementById("btn");
btn.addEventListener("click", () => {
  alert("Button Clicked!");
});
Output Preview:
Button shows alert when clicked

6. Creating & Removing Elements

  • Create elements with createElement.
  • Add using appendChild or insertBefore.
  • Remove with remove().
HTML Example:

<div id="container"></div>
JavaScript Example:

const newItem = document.createElement("p");
newItem.textContent = "Hello DOM!";
document.body.appendChild(newItem);
Output Preview:
New paragraph added below
