JavaScript ES6 Features

1. let and const

  • let is block-scoped variable.
  • const is block-scoped constant.
Example:

let a = 10;
const b = 20;
a = 15;
// b = 25; // Error
console.log(a, b);
Output Preview:
15 20

2. Arrow Functions

  • Shorter syntax for writing functions.
  • Does not have its own this.
Example:

const add = (x, y) => x + y;
console.log(add(5, 3));
Output Preview:
8

3. Template Literals

  • Use backticks ` ` for strings.
  • Supports variable interpolation: ${variable}.
Example:

const name = "Alice";
const message = `Hello, ${name}!`;
console.log(message);
Output Preview:
Hello, Alice!

4. Default Parameters

  • Functions can have default values for parameters.
Example:

function greet(name = "Guest") {
  console.log(`Hello, ${name}`);
}

greet("Bob");
greet();
Output Preview:
Hello, Bob
Hello, Guest

5. Destructuring

  • Extract values from arrays or objects into variables.
Example:

// Array destructuring
const numbers = [1,2,3];
const [x,y,z] = numbers;
console.log(x,y,z);

// Object destructuring
const person = {name: "Alice", age: 25};
const {name, age} = person;
console.log(name, age);
Output Preview:
1 2 3
Alice 25

6. Spread & Rest Operators

  • Spread: expands an array into elements.
  • Rest: collects remaining elements into array.
Example:

// Spread
const arr1 = [1,2];
const arr2 = [...arr1,3,4];
console.log(arr2);

// Rest
function sum(...nums) {
  return nums.reduce((a,b) => a+b, 0);
}
console.log(sum(1,2,3,4));
Output Preview:
[1,2,3,4]
10

7. Modules (import/export)

  • Use export to share variables/functions from a file.
  • Use import to bring them into another file.
Example:

// file: math.js
export const pi = 3.14;

// file: app.js
import {pi} from './math.js';
console.log(pi);
Output Preview:
3.14
