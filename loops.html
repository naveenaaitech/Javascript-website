JavaScript Loops

1. for Loop

  • The for loop repeats a block of code a specific number of times.
  • Used when you know how many times you want to run the loop.
  • Syntax: for(initialization; condition; increment/decrement)
Example:

for (let i = 1; i <= 5; i++) {
  console.log("Number " + i);
}
Output:
Number 1
Number 2
Number 3
Number 4
Number 5

2. while Loop

  • The while loop executes a block of code as long as the condition is true.
  • Used when you don’t know in advance how many times the loop should run.
  • Be careful: if the condition never becomes false, it creates an infinite loop.
Example:

let j = 1;
while (j <= 3) {
  console.log("Count " + j);
  j++;
}
Output:
Count 1
Count 2
Count 3

3. do...while Loop

  • The do...while loop executes the code block once before checking the condition.
  • This means it will always run at least one time, even if the condition is false.
Example:

let k = 1;
do {
  console.log("Step " + k);
  k++;
} while (k <= 3);
Output:
Step 1
Step 2
Step 3

4. for...of Loop

  • The for...of loop iterates through the values of an iterable like arrays or strings.
  • Introduced in ES6, it’s cleaner and easier than traditional for loops for arrays.
Example:

let fruits = ['Apple', 'Banana', 'Cherry'];

for (let fruit of fruits) {
  console.log(fruit);
}
Output:
Apple
Banana
Cherry

5. for...in Loop

  • The for...in loop iterates through the keys or properties of an object.
  • Commonly used to read all keys and values from an object.
Example:

let person = { name: 'Alice', age: 25, city: 'Delhi' };

for (let key in person) {
  console.log(key + ": " + person[key]);
}
Output:
name: Alice
age: 25
city: Delhi
