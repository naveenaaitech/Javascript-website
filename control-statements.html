JavaScript Control & Jumping Statements

1. if Statement

  • Executes a block of code only if the condition is true.
Example:

let age = 20;

if (age >= 18) {
  console.log("You are an adult");
}
Output Preview:
You are an adult

2. if...else Statement

  • Executes one block if the condition is true, otherwise another block.
Example:

let marks = 45;

if (marks >= 50) {
  console.log("Pass");
} else {
  console.log("Fail");
}
Output Preview:
Fail

3. if...else if...else Statement

  • Checks multiple conditions in sequence.
Example:

let score = 85;

if (score >= 90) {
  console.log("Grade A");
} else if (score >= 75) {
  console.log("Grade B");
} else {
  console.log("Grade C");
}
Output Preview:
Grade B

4. switch Statement

  • Used to select one code block to execute from multiple options.
Example:

let day = 2;

switch(day) {
  case 1:
    console.log("Monday");
    break;
  case 2:
    console.log("Tuesday");
    break;
  default:
    console.log("Other Day");
}
Output Preview:
Tuesday

5. What are Jumping Statements?

  • Jumping statements are used to change the normal flow of execution in a program.
  • They include break, continue, and return.

6. break Statement

  • Used to exit (or “break out” of) a loop or switch statement immediately.
Example:

for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    break; // exit loop when i is 3
  }
  console.log("Number: " + i);
}
Output Preview:
Number: 1
Number: 2

7. continue Statement

  • Skips the rest of the current loop iteration and continues with the next iteration.
Example:

for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue; // skip 3
  }
  console.log("Number: " + i);
}
Output Preview:
Number: 1
Number: 2
Number: 4
Number: 5

8. return Statement

  • Used inside a function to stop execution and return a value to the caller.
Example:

function add(a, b) {
  return a + b; // exits function and returns sum
}

let result = add(5, 3);
console.log(result);
Output Preview:
8
