JavaScript Window Object

1. What is Window Object?

  • The window object represents the browser window.
  • It is the global object in the browser.
  • All global variables and functions become properties of window.
Example Preview:
You can access browser features like alert, confirm, location, etc.

2. Alerts, Prompts & Confirms

  • alert(): displays a message.
  • prompt(): asks user for input.
  • confirm(): asks user to confirm an action.
Example:

function showAlert() {
  alert("Hello from window!");
}

function askName() {
  const name = prompt("Enter your name:");
  console.log("Name:", name);
}

function confirmAction() {
  const result = confirm("Do you want to proceed?");
  console.log("Confirmed:", result);
}
Output Preview:

3. Location Object

  • Provides information about current URL.
  • Can be used to redirect or reload the page.
Example:

console.log("Current URL:", window.location.href);

function goToGoogle() {
  window.location.href = "https://www.google.com";
}

function reloadPage() {
  window.location.reload();
}
Output Preview:

4. Timers (setTimeout & setInterval)

  • setTimeout(): run code after delay.
  • setInterval(): run code repeatedly.
Example:

setTimeout(() => console.log("Hello after 2s"), 2000);

let counter = 0;
const interval = setInterval(() => {
  console.log("Counter:", counter);
  counter++;
  if(counter > 2) clearInterval(interval);
}, 1000);

5. Console Object

  • Used for debugging and logging information.
  • Common methods: log(), warn(), error(), table().
Example:

console.log("Hello Console");
console.warn("This is a warning");
console.error("This is an error");
console.table([{name:"Alice", age:25}, {name:"Bob", age:30}]);

6. Window Size

  • window.innerWidth and window.innerHeight give viewport size.
Example:

console.log("Width:", window.innerWidth);
console.log("Height:", window.innerHeight);

Check console to see window width and height

7. Window Name

  • window.name stores a string for the current window.
  • It persists across page reloads for the same tab.
  • You can get or set it with window.name.
Example:

console.log("Current window name:", window.name);

function setWindowName() {
  const newName = prompt("Enter window name:");
  window.name = newName;
  console.log("Window name set to:", window.name);
}

function getWindowName() {
  console.log("Window name is:", window.name);
}
