Closures allow maintaining state between function calls.

2. How Closures Work

  • Inner function remembers variables from the outer function.
Example:

function outer() {
  let count = 0;
  function inner() {
    count++;
    console.log(count);
  }
  return inner;
}

const fn = outer();
fn(); // 1
fn(); // 2
fn(); // 3

3

3. Practical Example

  • Closures are useful for creating private variables.
Example:

function createCounter() {
  let counter = 0;
  return function() {
    counter++;
    console.log(counter);
  };
}

const counter1 = createCounter();
counter1(); // 1
counter1(); // 2
Output Preview:
1
2

4. Using Closures for Data Privacy

  • Variables inside closure are private and cannot be accessed directly from outside.
Example:

function secret() {
  let privateVar = "secret";
  return function() {
    console.log(privateVar);
  };
}

const reveal = secret();
reveal(); // secret
Output Preview:
secret

5. Closures in Loops

  • Closures help retain loop variable value inside asynchronous callbacks.
Example:

for(let i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 1000);
}
