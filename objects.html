JavaScript Objects

1. What is an Object?

  • Objects are collections of key-value pairs.
  • Used to store structured data.
Example:

const person = {
  name: "Alice",
  age: 25,
  city: "New York"
};
console.log(person);
Output Preview:
{ name: "Alice", age: 25, city: "New York" }

2. Accessing Properties

  • Use dot notation: object.key
  • Use bracket notation: object["key"]
Example:

console.log(person.name);   // Dot notation
console.log(person["age"]); // Bracket notation
Output Preview:
Alice
25

3. Modifying Properties

  • Change the value of an existing property.
Example:

person.age = 26;
console.log(person.age);
Output Preview:
26

4. Adding & Deleting Properties

  • Add new property: object.key = value
  • Delete property: delete object.key
Example:

// Add
person.country = "USA";
console.log(person.country);

// Delete
delete person.city;
console.log(person);
Output Preview:
USA
{ name: "Alice", age: 26, country: "USA" }

5. Object Methods

  • Functions inside objects are called methods.
Example:

person.greet = function() {
  console.log("Hello, " + this.name);
};

person.greet();
Output Preview:
Hello, Alice

6. this Keyword in Objects

  • this refers to the object calling the method.
Example:

const obj = {
  name: "Bob",
  greet: function() {
    console.log("Hi, " + this.name);
  }
};
obj.greet();
Output Preview:
Hi, Bob

7. Object.keys(), Object.values(), Object.entries()

  • Object.keys() → returns all property names.
  • Object.values() → returns all property values.
  • Object.entries() → returns key-value pairs as arrays.
Example:

const keys = Object.keys(person);
const values = Object.values(person);
const entries = Object.entries(person);

console.log(keys);
console.log(values);
console.log(entries);
Output Preview:
[ "name", "age", "country" ]
[ "Alice", 26, "USA" ]
[ ["name", "Alice"], ["age", 26], ["country", "USA"] ]

8. hasOwnProperty()

  • Checks if an object has a specific property.
Example:

console.log(person.hasOwnProperty('age'));   // true
console.log(person.hasOwnProperty('salary')); // false
Output Preview:
true
false
