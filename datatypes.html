JavaScript Data Types

1. String

  • Represents textual data enclosed in quotes (" " or ' ').
  • Used for words, sentences, or characters.
Example:

let name = "Alice";
console.log(name);
Output Preview:
Alice

2. Number

  • Represents numeric values (integer or decimal).
  • No separate type for integers and floats in JavaScript.
Example:

let age = 30;
let price = 99.99;
console.log(age, price);
Output Preview:
30 99.99

3. Boolean

  • Represents logical values: true or false.
  • Used in conditions and comparisons.
Example:

let isStudent = true;
console.log(isStudent);
Output Preview:
true

4. Undefined

  • A variable declared but not assigned has the value undefined.
Example:

let x;
console.log(x);
Output Preview:
undefined

5. Null

  • Represents an empty or unknown value.
  • Explicitly set by the programmer.
Example:

let y = null;
console.log(y);
Output Preview:
null

6. Object

  • Used to store collections of data in key-value pairs.
Example:

let person = {
  name: "Bob",
  age: 25
};
console.log(person.name, person.age);
Output Preview:
Bob 25

7. Array

  • Special type of object used to store multiple values.
  • Values are ordered and accessed by index (starting from 0).
Example:

let colors = ["red", "green", "blue"];
console.log(colors[0], colors[2]);
Output Preview:
red blue

8. Symbol

  • Introduced in ES6.
  • Represents a unique and immutable identifier.
Example:

let id = Symbol("uniqueId");
console.log(id);
Output Preview:
Symbol(uniqueId)

9. BigInt

  • Used to represent integers larger than Number.MAX_SAFE_INTEGER.
  • Created by appending n at the end of the number.
Example:

let bigNum = 123456789012345678901234567890n;
console.log(bigNum);
Output Preview:
123456789012345678901234567890n
