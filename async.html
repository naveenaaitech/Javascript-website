JavaScript Async & Sync

1. Synchronous JavaScript

  • Runs code sequentially, one after another.
  • Each line waits for the previous line to finish.
Example:

console.log("Step 1");
console.log("Step 2");
console.log("Step 3");
Output Preview:
Step 1
Step 2
Step 3

2. What is Asynchronous JavaScript?

  • Allows code to run without blocking the main thread.
  • Examples: API calls, timers, reading files.
Example Preview:
Async code allows other code to run while waiting for results.

3. setTimeout & setInterval

  • setTimeout: runs function after delay.
  • setInterval: runs function repeatedly at interval.
Example:

setTimeout(() => console.log("Hello after 2s"), 2000);

let count = 0;
const interval = setInterval(() => {
  console.log("Count:", count);
  count++;
  if(count > 2) clearInterval(interval);
}, 1000);
Output Preview:
Hello after 2s
Count: 0
Count: 1
Count: 2

4. Promises

  • Represents future result of async operation.
  • Methods: then, catch.
Example:

const promise = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Success!"), 2000);
});

promise
  .then(result => console.log(result))
  .catch(error => console.error(error));
Output Preview:
Success! (after 2s)

5. async / await

  • async marks function as asynchronous.
  • await waits for a promise to resolve.
Example:

async getData() {
  const promise = new Promise((resolve) => {
    setTimeout(() => resolve("Data fetched!"), 2000);
  });
  const result = await promise;
  console.log(result);
}

getData();
Output Preview:
Data fetched! (after 2s)

6. Fetch API with async/await

  • Use async/await with fetch for cleaner code.
Example:

const btn = document.getElementById("fetchBtn");
btn.addEventListener("click", async () => {
  try {
    const response = await fetch("https://jsonplaceholder.typicode.com/todos/1");
    const data = await response.json();
    document.getElementById("output").textContent = JSON.stringify(data);
  } catch(error) {
    console.error("Error:", error);
  }
});
Output Preview:
Click button to fetch data asynchronously

7. Error Handling

  • Use try/catch to handle async errors gracefully.
Example:

async fetchInvalid() {
  try {
    const res = await fetch("https://jsonplaceholder.typicode.com/invalid");
    const data = await res.json();
    console.log(data);
  } catch(err) {
    console.log("Error occurred:", err);
  }
}

fetchInvalid();
Output Preview:
Error occurred: [Error details]
