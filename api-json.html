JavaScript API & JSON

1. What is an API?

  • API (Application Programming Interface) allows communication between apps.
  • Web APIs provide data over HTTP in formats like JSON or XML.
Example Preview:
API allows fetching data from servers to your webpage.

2. What is JSON?

  • JSON (JavaScript Object Notation) is a lightweight data format.
  • Used to exchange data between server and client.
Example:

const person = {
  name: "Alice",
  age: 25
};
console.log(JSON.stringify(person));  // Convert object to JSON string
Output Preview:
{"name":"Alice","age":25}

3. Fetching Data from API

  • Use fetch(url) to get data from an API.
  • Returns a promise.
Example:

const btn = document.getElementById("fetchBtn");
btn.addEventListener("click", () => {
  fetch("https://jsonplaceholder.typicode.com/todos/1")
    .then(response => response.json())
    .then(data => document.getElementById("output").textContent = JSON.stringify(data))
    .catch(error => console.error("Error:", error));
});
Output Preview:
Click the button to fetch data from API

4. Handling Errors

  • Use catch to handle network or parsing errors.
Example:

fetch("https://jsonplaceholder.typicode.com/invalid")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.log("Error occurred:", error));
Output Preview:
Error occurred: [Error details]
