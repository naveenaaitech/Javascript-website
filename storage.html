JavaScript Web Storage

1. What is Web Storage?

  • Web Storage allows storing data in the browser.
  • Two types: localStorage and sessionStorage.
Example Preview:
Data can be stored and retrieved in the browser without a server.

2. localStorage

  • Stores data with no expiration.
  • Accessible even after closing and reopening the browser.
Example:

localStorage.setItem("name", "Alice");
console.log(localStorage.getItem("name")); // Alice
Output Preview:
Alice

3. sessionStorage

  • Stores data for the session only.
  • Data is cleared when the tab or browser is closed.
Example:

sessionStorage.setItem("sessionName", "Bob");
console.log(sessionStorage.getItem("sessionName")); // Bob
Output Preview:
Bob

4. Removing Data

  • Remove a specific item using removeItem.
  • Clear all items using clear().
Example:

localStorage.setItem("city", "New York");
console.log(localStorage.getItem("city")); // New York

localStorage.removeItem("city");
console.log(localStorage.getItem("city")); // null

localStorage.setItem("country", "USA");
localStorage.clear(); // removes all items
Output Preview:
New York
null

5. Storing Objects

  • Objects must be converted to JSON string before storing.
  • Use JSON.stringify() to store and JSON.parse() to retrieve.
Example:

const user = {name: "Alice", age: 25};
localStorage.setItem("user", JSON.stringify(user));

const retrievedUser = JSON.parse(localStorage.getItem("user"));
console.log(retrievedUser.name); // Alice
Output Preview:
Alice
