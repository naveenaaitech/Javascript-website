JavaScript Events

1. What are Events?

  • Events are actions that occur on HTML elements, like clicks, keypresses, or mouse movements.
Example Preview:
Click, type, or hover on elements to trigger events.

2. Inline Events

  • Add event directly in HTML using attributes like onclick.
Example:

<button onclick="alert('Button Clicked!')">Click Me</button>
Output Preview:

3. Using addEventListener

  • Attach event listener via JavaScript.
  • Supports multiple events on same element.
Example:

const btn = document.getElementById("btn");
btn.addEventListener("click", () => {
  alert("Clicked using addEventListener!");
});
Output Preview:

4. Common Event Types

  • click: when element is clicked
  • mouseover / mouseout: when hovering over element
  • keydown: when a key is pressed
  • submit: when form is submitted
Example:

const box = document.getElementById("box");
box.addEventListener("mouseover", () => box.style.backgroundColor = "yellow");
box.addEventListener("mouseout", () => box.style.backgroundColor = "lightblue");
Output Preview:

5. Event Object & preventDefault

  • The event object gives info about the event.
  • Use preventDefault() to stop default behavior.
Example:

const form = document.getElementById("form");
form.addEventListener("submit", (e) => {
  e.preventDefault();
  alert("Form submitted but page did not reload!");
});
Output Preview:

6. Multiple Events

  • You can attach multiple events to the same element using multiple addEventListener.
Example:

const multiBtn = document.getElementById("multiBtn");
multiBtn.addEventListener("mouseover", () => multiBtn.style.backgroundColor = "green");
multiBtn.addEventListener("mouseout", () => multiBtn.style.backgroundColor = "orange");
multiBtn.addEventListener("click", () => alert("Button Clicked!"));
