JavaScript Arrays

1. What is an Array?

  • An array is a collection of values stored in a single variable.
  • Values can be of any type.
Example:

const fruits = ["Apple", "Banana", "Orange"];
console.log(fruits);
Output Preview:
["Apple", "Banana", "Orange"]

2. Accessing Array Elements

  • Use index numbers starting from 0.
Example:

console.log(fruits[0]); // Apple
console.log(fruits[2]); // Orange
Output Preview:
Apple
Orange

3. Modifying Array Elements

  • Change a value by its index.
Example:

fruits[1] = "Mango";
console.log(fruits);
Output Preview:
["Apple", "Mango", "Orange"]

4. Array Methods

  • push(): adds element at the end.
  • pop(): removes last element.
  • shift(): removes first element.
  • unshift(): adds element at the beginning.
  • indexOf(): finds index of element.
  • splice(): add/remove elements.
  • slice(): copy part of array.
Example:

// push & pop
fruits.push("Pineapple");
console.log(fruits);
fruits.pop();
console.log(fruits);

// shift & unshift
fruits.shift();
fruits.unshift("Strawberry");
console.log(fruits);

// indexOf
console.log(fruits.indexOf("Orange"));

// splice (remove 1 element at index 1)
fruits.splice(1, 1);
console.log(fruits);

// slice (copy index 0 to 1)
const newFruits = fruits.slice(0, 2);
console.log(newFruits);
Output Preview:
["Apple", "Banana", "Orange", "Pineapple"]
["Apple", "Banana", "Orange"]
["Strawberry", "Banana", "Orange"]
2
["Strawberry", "Orange"]
["Strawberry", "Orange"]

5. Iterating over Arrays

  • Use for, for...of, forEach to loop through arrays.
Example:

// for loop
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

// for...of
for (const fruit of fruits) {
  console.log(fruit);
}

// forEach
fruits.forEach((fruit) => console.log(fruit));
Output Preview:
Strawberry
Orange
Strawberry
Orange
Strawberry
Orange
